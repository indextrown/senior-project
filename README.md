## 프로젝트 소개
각종 SNS에서 생일 카페 스케줄을 자동으로 수집하여, 이를 바탕으로 중개 플랫폼 시스템을 개발하는 것이다.

## ⏲️ 개발 기간 
- 2024.03.01 ~ 2024.00.00

## 🧑‍🤝‍🧑 개발자 소개 
- **송민석** : 팀장, Spring Web 개발
- **김동현** : 크롤링 데이터 수집, Server 개발
- **장태현** : 데이터 분석 및 자연어 처리
- **최윤진** : 데이터베이스 개발 및 유지보수

## ⚙️ 기술 스택
- **Server** : AWS EC2
- **Crawling** : Python
- **DB** : Mysql
- **아이디어 회의** : Discord, Github
